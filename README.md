# cpp-student-class-project
A simple C++ OOP program that demonstrates how to create a Student class, create objects, store student information, and display the details using classes and objects.

# Student Class Project in C++

## Description
This is a simple C++ Object-Oriented Programming (OOP) project.

The program demonstrates:
- Creating a class named `student`
- Declaring data members (`name`, `age`, `marke`)
- Creating multiple objects (`s1`, `s2`, `s3`)
- Storing values in objects
- Displaying student information using `cout`

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
